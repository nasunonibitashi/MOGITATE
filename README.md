# MOGITATE(もぎたて）

## 環境構築

**Dockerビルド**
1. 'git clone git@github.com:nasunonibitashi/mogitate.git'
2. 'docker-compose up -d --build'

**Laravel環境構築**
1. 'docker-compose exec php bash'
2. 'composer install'


**URL**
- 開発環境:http://localhost/
- phpMyAdmin:http://localhost:8080/

**使用技術（実行環境）**
- PHP8.3.0
- Laravel8.83.27
- MySQL8.0.26
